## Notes for maintainers / contributors

* Styling lives in `style.css`. AI widget styling & logic are included in `ai-widget` blocks inside `index.html` for simplicity.
* Use the `:root` CSS variables to tune colors (accent1, accent2, text, glass) to match branding.
* Keep the `<meta name="viewport" content="width=device-width, initial-scale=1.0">` tag in `<head>` for mobile responsiveness.
